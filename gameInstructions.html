Let's play!

Instructions

  • The player begins with 3 guesses.
  • Once the game begins, there will be a set of dashes shown in an alert window, where each dash represents a letter in the word.
  • In other words, there will be x number of dashes, where x represents the number of letters in the word
  • If the player are able to correctly guess a letter in the word, the blanks with that letter will be filled in the right places
  • However, if the player are unable to do so, you will lose a guess.
